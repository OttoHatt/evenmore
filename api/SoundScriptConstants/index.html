SoundScriptConstants

This item only works when running on the client. Client

This modules defines the format of SoundScripts, and a few generic templates that you can use in your own games. SoundScripts in this module are automatically loaded; simply put the name of one (i.e. city) in a tagged trigger to use it.

If you create a SoundScript with this system, please consider submitting a PR! This module is most valuable as a library of soundscapes that can be used in a wide variety of games.

Format inspired by the Source Engine soundscape system.

SoundValue​

</>
type SoundValue = {number} | number | nil

This is a low-level property for a soundscript. These are evaluated each delay a sound is played.

  • A number in -> The same number out
  • A two-long number array in -> A random value within the range of the two
  • A 'nil' in -> a 'nil' out, falling back to a default value (i.e. volume = 1)
